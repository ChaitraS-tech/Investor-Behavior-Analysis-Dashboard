#  Investor Behavior Analysis Dashboard (Power BI)

# Project Overview

This project focuses on analyzing investor behavior using Microsoft Power BI to derive meaningful insights. The dashboard provides a clear visualization of investor demographics, preferred investment avenues, and decision-making patterns to support better financial strategies.

---

# Objectives

* Analyze investor demographics and profiles
* Identify preferred investment options
* Study investment monitoring frequency
* Understand key factors influencing investment decisions
* Build an interactive and insightful dashboard

---

# Tools & Technologies Used

* Microsoft Power BI (Data Visualization)
* Power Query (Data Transformation)
* Excel / CSV (Dataset)

---

# Dataset Description

The dataset includes the following fields:

* Age Group
* Gender
* Occupation
* Investment Avenue
* Investment Duration
* Monitoring Frequency
* Expected Returns
* Risk Appetite
* Investment Purpose

---

# Data Processing

1. Imported raw dataset into Power BI
2. Performed data cleaning using Power Query
3. Applied transformations such as:

   * Unpivoting columns
   * Handling missing values
   * Standardizing categorical values
4. Created calculated columns and measures for analysis

---

# Dashboard Features

* KPI Cards:

  * Total Investors
  * Average Investment Duration
  * Most Preferred Investment Type
  * Average Expected Returns

* Visualizations:

  * Bar Chart (Investment by Age Group)
  * Pie Chart (Investment Distribution)
  * Column Chart (Returns vs Risk)
  * Stacked Chart (Occupation vs Investment Type)
  * Slicers for dynamic filtering

* Filters (Slicers):

  * Age Group
  * Gender
  * Occupation
  * Investment Type

---

# Key Insights

* Identified most preferred investment options among different age groups
* Analyzed how risk appetite affects investment decisions
* Observed trends in investment duration and monitoring frequency
* Highlighted key motivations behind financial decisions

---

# How to Run the Project

1. Download the `.pbix` file
2. Open in Microsoft Power BI Desktop
3. Load the dataset if required
4. Interact with dashboard using filters and slicers
